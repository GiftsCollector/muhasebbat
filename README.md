# نظام محاسبات للمقاولات

تطبيق ويب بسيط بلغة Python و Flask لإدارة:
- دليل الحسابات المرتبط بالمشاريع وبنود الأعمال
- إدارة المشاريع (كود المشروع، العميل، قيمة العقد، بنود الأعمال)
- إنشاء المستخلصات وربطها بالمشاريع ومقاولي الباطن
- قيود محاسبية تلقائية عند اعتماد المستخلص
- إدارة مقاولي الباطن والعمالة والمعدات

## المتطلبات

- Python 3.11 أو أحدث
- تثبيت الحزم من `requirements.txt`

## التشغيل

1. افتح موجه الأوامر في المجلد `محاسبات`
2. أنشئ بيئة افتراضية (اختياري):
   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   ```
3. ثبّت المكتبات:
   ```powershell
   python -m pip install -r requirements.txt
   ```
4. شغّل التطبيق:
   ```powershell
   python app.py
   ```

5. افتح المتصفح وزُر:
   `http://127.0.0.1:5000`

## ملاحظات

- سيُنشأ ملف قاعدة البيانات `data.db` تلقائيًا عند أول تشغيل.
- استخدم صفحة `دليل الحسابات` لإضافة الحسابات الأساسية مثل الأصول، الالتزامات، الإيرادات، المصروفات.
- صفحة `المستخلصات` تنشئ قيدًا محاسبيًا تلقائيًا مرتبطًا بالمشروع.
- يدعم النظام الآن الموردين، أوامر الشراء، وحركات المخزون لكل مشروع.
- أوامر الشراء تنشئ قيدًا محاسبيًا تلقائيًا عند الحفظ إذا كانت حسابات `مواد` و`موردين` موجودة.
- حركات المخزون تضيف قيودًا محاسبية تلقائيًا عند الاستلام أو الصرف باستخدام حسابات `مواد`، `أصول`، `موردين`، و`المصروفات`.
- أضفنا صفحة `محاسبة التكاليف` لتسجيل تكاليف مباشرة وغير مباشرة ومراقبة تكلفة البنود.
- صفحة `تقرير التكلفة` تعرض ملخصًا لتكلفة المشاريع وتكلفة الوحدة لكل بند.
- صفحة `تقرير المخزون` تعرض تقييم المخزون بنظامي `Average` و`FIFO`.
